# Product Reviews Sentiment Analysis

## Project Overview
This project performs end-to-end sentiment analysis on  product reviews to understand customer satisfaction. The project includes:

- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA) with business insights
- Sentiment prediction using ML models

## Dataset
- Source: [Kaggle - Flipkart Product Customer Reviews Dataset](https://www.kaggle.com/datasets/niraliivaghani/flipkart-product-customer-reviews-dataset)
- Contains product reviews, ratings, and sentiment labels (positive, neutral, negative)
